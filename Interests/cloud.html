Cloud computing is defined as:
the practice of using a network of remote servers
hosted on the Internet to store, manage, and process
data, rather than a local server or a personal computer.
How Cloud services affect me:
I find cloud services extremely useful in my every day life.
I used cloud 9, a Platform as a service(Paas) in the creation
of this website. I use gmail for formal communication. I use icloud
with my iPhone. I find icloud extremely useful as you can set automatic
back ups. It is extremely useful for storage.
My favourite cloud service:
My favourite cloud service is Gmail.
I use Gmail every day to send and recieve emails.
I use it for work,college and socially. Gmail is a Communication as a service (Caas).
I find Gmail extremely useful as a means of formal communication
I receive my weekly pay check via email and I have used Gmail to communicate with various employers
